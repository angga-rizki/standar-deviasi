# standar-deviasi
Aplikasi web sistem informasi untuk pehitungan nilai standar deviasi. Di dalam web, input dilakukan dengan meng-upload file excel berisi nilai - nilai variabel yang akan dihitung.

Default akun :
- username: nia
- password: nia

CodeIgniter4
MySQL

Library:
Bootstrap 4.5.0
DataTables 1.10.21
jQuery 3.5.1
spreadsheet-reader-0.5.11
